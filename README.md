# Window 10 - Phân tích và Tái tạo khuôn mặt (PCA & SVD)

Dự án môn học Python: Ứng dụng web phân tích, nén và tái tạo ảnh khuôn mặt sử dụng thuật toán PCA (Principal Component Analysis) và SVD (Singular Value Decomposition).

## Yêu cầu hệ thống
* **Python**: Phiên bản **3.12**.

## Hướng dẫn cài đặt
1. Clone repo:
  bash
  git clone <link>
2. Tạo môi trường ảo:
  python -m venv venv
  venv\Scripts\activate
3. Cài thư viện và chạy ứng dụng:
  pip install -r requirements.txt
  python app.py
Truy cập đường dẫn: http://127.0.0.1:8050/ trên trình duyệt.
